Old news

  • 13 Jul 2008 There will likely not be a localized version of GrandPerspective 0.9.12. The reason is that I have not received any updated localizations. The fairly high release rate of GrandPerspective may have been wearing existing localizers out a bit. To give them some well-earned rest, I have not actively chased them for a change. I am now instead aiming for a localized Version 1.00 of GrandPerspective in as many languages as possible.

    Of course, should I still receive localizations for Version 0.9.12, or if anyone is interested in a localized release of GrandPerspective with only the Dutch language (laat het mij dan weten), a localized release of GrandPerspective 0.9.12 may still appear.

  • 14 Jun 2008 GrandPerspective 0.9.12.1 is available for download. It fixes a bug in 0.9.12 that could cause the application to crash when using a Path test applied to folders in a filter.

  • 10 Jun 2008 GrandPerspective 0.9.12 has been released and is available for download. It includes three eagerly awaited features: correct handling of hard-links (scan your Time Machine back-ups), optional showing of package contents, and the ability to apply mask and filter rules to folders.

  • 22 Apr 2008 A localized version of GrandPerspective 0.9.11 is now available for download. Unfortunately, it contains fewer localizations than the previous release, mostly because various localizers were too busy to provide an update in their language. Maybe you can help out for the next release? Furthermore, thanks to all who have shown support by making a donation. Thanks to a recent generous donation the upgrade to Leopard is nearly covered (the dollar being a bit weak doesn't help unfortunately) so expect improved support for OS X 10.5 in the next release.

  • 17 Apr 2008 GrandPerspective has been downloaded more than 100.000 times. Happy happy joy joy.

  • 18 Mar 2008 GrandPerspective 0.9.11 has been released and is available for download. It includes two long-awaited features: deletion of files and folders from within the application, and the ability to color files by their file type.

  • 20 Jan 2008 A localized version of GrandPerspective 0.9.10 is now available for download. Sorry that it has taken slightly longer than usual, but that was mostly due to the holidays.

  • 12 Dec 2007 GrandPerspective 0.9.10 has been released. Newly added is the ability to select folders in the view, and the ability to show the entire volume (including free space). There have been a few other improvements as well. By the way, don't be confused by the new version numbering scheme. This release is the successor of Version 0.99 (which would be called 0.9.9 under the new naming scheme). I changed the naming scheme as I do not yet consider this Version 1.00.

  • 24 Jul 2007 GrandPerspective has been downloaded more than 50.000 times. Great! However, I should apologise for the recent lull in activities. I have just been too busy lately. Nevertheless, Version 1.00 is in the making and will appear sometime. Patience, my friends...

  • 18 Apr 2007 Thanks to lifehacker.com for featuring GrandPerspective as download of the day. It has resulted in another spike in the download stats.

  • 28 Mar 2007 A localized version of GrandPerspective 0.99 is now available for download.

  • 24 Feb 2007 GrandPerspective 0.99 has been released. It is the first time the application has been officially released as a Universal Binary (many thanks to Martin Kahr for providing unofficial Universal Binaries for earlier releases). Furthermore, this latest release offers a choice of file size measures and includes help documentation. It also includes a few minor improvements and bug fixes. Don't linger, just download.

  • 3 Feb 2007 The GrandPerspective project has received its first donation! Many thanks to the anonymous donor. While on the subject, feel free to make a donation yourself to show your appreciation and encourage further development of GrandPerspective.

  • 9 Jan 2007 A localized version of GrandPerspective 0.98 is now available. You can download it from the official release area. By the way, localizations in additional languages are still greatfully accepted.

  • 15 Dec 2006 GrandPerspective 0.98 has been released. The main new feature is that user created masking and filtering rules are now automatically stored when the application terminates, so that they are also available the next time. Quite a few other improvements as well, so worth a download I'd say. But then again, I would say that, wouldn't I?

  • 10 Dec 2006 A localized version of GrandPerspective 0.97 is now available. It is the first localized GrandPerspective release and includes translations into Japanese, German, and Dutch. Download it from the official release area if you are interested. Is your favourite language still missing? Maybe you can provide it yourself.

  • 14 Nov 2006 GrandPerspective 0.97 has been released. It offers a choice of palettes now, all of which more aesthetically pleasing than the original one. You can now also set a few preferences. Go ahead, download it, and feast your eyes. This version is also fully internationalized. So please help out with localization if you can.

  • 12 Aug 2006 GrandPerspective 0.96 has been released. It supports "filtered" views, and the GUI has been streamlined as well. Don't be shy, download it and give filtering a try.

  • 5 Jun 2006 GrandPerspective 0.95 has been released. It includes the ability to "mask" files in the view. This is handy for quickly spotting certain types of files. Download the latest release and give masking a go.

  • 10 May 2006 There is now an unofficial Universal Binary for GrandPerspective available, thanks to Martin Kahr. See the Download section for details.

  • 9 May 2006 GrandPerspective 0.94 has been released. It includes the new "re-scan" command, which can generate up-to-date views after you have cleaned up your file system a bit. Furthermore, more minor changes to the GUI, as well as to the source code. Download and enjoy.

  • 14 Apr 2006 After noticing a surge in downloads of GrandPerspective I found out that it had been featured as download of the day on O'Reilly's MacDevCenter.com. Great! By the way, I can generally recommend O'Reilly books, two of which I actually used to develop GrandPerspective.

    • My apologies for the recent lull in GrandPerspective's further development. I have been busy with other things lately, but expect another release soon, hopefully as a Universal Binary.

  • 22 Jan 2006 GrandPerspective 0.93 has been released. It features the most requested feature enhancement so far, the ability to open files in Finder. On top of that, you can now save an image of the directory tree to file. Why are you still reading this? Just download it.

  • 10 Jan 2006 GrandPerspective 0.92 has been released. I finally had some time to code again, and this new release is the result. It features the new Twin Window command, an improved Duplicate Window command and includes two bug fixes. So go ahead and download it.

  • 9 Oct 2005 Sorry, don't expect any new release of GrandPerspective for the next two months or so. Although I am keen to extend it further, I'm in the process of moving house, which will suck up most of my spare time in the forseeable future. Just so you know...
