How to scan a package

You can use GrandPerspective to scan the contents of a single package, for example, your Photos library. You can do so as follows:

  1. Configure GrandPerspective to show package contents by default:

    GrandPerspective → Preferences... → Display → Enable show package contents
  2. You are now able to select a package when prompted to choose a folder to scan:

    File → Scan Folder...
