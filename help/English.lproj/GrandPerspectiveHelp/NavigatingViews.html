Navigating views

You can focus on files in a view window by hovering over it with your mouse. Clicking on a file locks the focus so that it does not change when you move the mouse. Click once more, anywhere in the view, to unlock the focus. When the focus is locked, you can change the folder that is shown in the view. By clicking the Down button, you descend into the first subfolder on the path to the selected file. You can keep going down, until you've reached the file itself. Clicking Up will take you back up, one level at the time.

When the focus is locked, you can show the selected file in Finder using the Reveal button. There you can examine the file, rename it, delete it, etc.
