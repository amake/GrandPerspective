The view's Info panel

The Info panel in the view's drawer gives information about the folder that has been scanned. It shows the volume on which the folder resides, the name of the folder, and the time when it was scanned. The file size measure that has been used is shown as well. When a filter has been applied, it will also show details of this. You can edit the text in this text view, which will be stored when you save the scan data. You can use this to associate useful information with the scan.

The panel also shows the following five sizes:

  • Volume size: The total size of the volume.
  • Size of scanned files: The total size of the scanned files, using the specified file size measure and after having applied the filter (if any).
  • Initial free space: The available space on the volume, as reported by the Operating System at the time of the scan.
  • Miscellaneous used space: All used space that is not accounted for by the size of the scanned files. This includes the following:
    • Files that were not scanned, either because the scanned folder was not at the root of the volume, or the user did not have permissions to access all subfolders.
    • Files that are excluded by the filter (if any).
    • Disk space that is not accounted for because the Logical file size measure is used, which underestimates the space that files actually take up.
    • Low-level file system data, needed to store amongst others the folder hierarchy.
  • Freed space: The total size of files that have been deleted from the scanned folder since the scan was made. It includes only files that have been deleted using the view's Delete button (or the Delete button of a twinned or duplicated view).
