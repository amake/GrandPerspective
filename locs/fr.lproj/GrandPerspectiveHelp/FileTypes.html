Type de fichiers

GrandPerspective utilise le système Uniform Type Identifier qui est partie intégrante de Mac OS X pour identifier les types de fichiers. Un fichier est associé à un type de fichier grâce à son extention. Par exemple, les fichiers terminant par .jpg ou .jpeg sont associés au type de fichier public.jpeg qui représente les images JPEG. Un type de fichier peut se conformer à d'autres types de fichiers (plus généraux). Par exemple, les fichiers du type public.jpeg se conforment au type plus général public.image.

Vous pouvez colorer les fichiers selon leur type de fichier en séléctionnant la palette de couleurs "Type de fichiers" depuis le panneau d'affichage. Vous pouvez changer la manière dont les types de fichiers sont représentés par différentes couleurs en éditant la palette "Type de fichiers".

Note : Les "Uniform Type" (Type Uniforme) ont été introduits dans Mac OS X 10.3 mais sont sérieusement supportés depuis Mac OS X 10.4 et supérieurs. Dans Mac OS X 10.3, il n'y a qu'un petit nombre de types de fichiers qui sont présents, de nombreux fichiers seront classés en "type de fichier inconnu".
