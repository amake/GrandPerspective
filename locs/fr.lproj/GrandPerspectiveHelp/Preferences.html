Préférences

L'application peut être personnalisée en utilisant les Préférences, elles sont accessibles depuis le menu de GrandPerspective. Le panneau de Préférences est divisé en trois parties : Actions, Info et Affichage.

Dans la section Actions vous pouvez configurer si la suppression de fichier est supporté dans la fenêtre principale ou non. Vous pouvez activer/désactiver la suppression des fichiers seuls ou l'activer pour des dossiers entiers. De plus, vous pouvez spécifier si une confirmation est requises ou non.

Attention : La suppression de dossier déplacera le dossier entier vers la Corbeille, cela inclut tous les fichiers présent dans le dossier qui ne sont pas visible dans la vue. Un fichier peut ne pas être visible car il est masqué ou filtré, il n'a pas pu être analysé pour cause de permissions insuffisantes ou il est apparu dans le dossier après l'analyse de ce dernier. En bref, vous devez faire très attention quand vous supprimez un dossier.

Dans la partie Info vous pouvez configurer la mesure de la taille des fichiers qui sera utilisé lors des analyses. Un changement de ce paramètre ne prendra effet lorsque une nouvelle analyse sera effectuée. Il n'affecte pas les vues existantes ou les vues qui ont été créées après une nouvelle analyse (les paramètres seront pris depuis la vue originelle). La mesure de la taille des fichiers qui est utilisée par une fenêtre spécifique peut être trouvée dans le panneau Info.

La partie Affichage contient les paramètres qui touchent à l'apparence de la vue. Vous pouvez spécifier les palettes de couleurs et le schéma de colorisation qui seront utilisés pour toutes nouvelles vues. En éditant le schéma de couleurs "Type de fichiers" vous pouvez contrôler les couleurs qui sont liées à certains types de fichiers. Les changements de ce paramètre sera prise en compte dans toutes les vues qui utilise la colorisation par type de fichiers. Vous pouvez également spécifier si le contenu des paquets doit être visible pour les nouvelles vues crées ou non. La plupart de ces paramètres est seulement spécifique à l'apparence de la vue initialement crée. Vous pouvez ultérieurement changer l'apparence d'une vue en utilisant le panneau Affichage.
