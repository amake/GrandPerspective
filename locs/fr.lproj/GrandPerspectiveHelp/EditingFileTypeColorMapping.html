Editez le schéma de couleurs "Type de fichiers"

Depuis les Préférences vous pouvez editer le schéma de couleurs "Type de fichiers". Vous pouvez le faire en changeant l'ordre des types de fichiers qui sont utilisés par le schéma de couleurs. Chaque fichier correspond à une couleur déterminant le premier type de fichier dans la liste des types auxquels correspond le fichier. L'ordre des types de fichiers est important pour deux raisons :

  • De nombreux fichiers se conforment à plus d'un type de fichiers : un type spécifique et probablement un ou plusieurs types généraux. L'ordre détermine alors lequel de ces types sera utilisé pour colorer le fichier. Par exemple, si le type public.jpeg apparaît en dessous du type plus général public.image alors les images JPEG seront assignées à la même couleur que les autres fichiers d'image. D'un autre côté, si le type public.jpeg apparaît au dessus du type public.image alors les images JPEG seront colorées de manière différente des autres fichiers d'image.
  • Le nombre de couleurs disponibles est typiquement plus petit que le nombre de types de fichiers. Il n'est toutefois pas impossible d'assigner une couleur à chacun des types. Seuls les types dans le haut de la liste peuvent avoir leur propre couleur. Les types "inférieurs" peuvent tous correspondre à la dernière couleur qui représente tout les autres types de fichiers.

Si un type de fichiers apparaît au dessous d'un type de fichiers plus général auquel il se conforme, il est "caché" car il ne sera jamais utilisé. Le type de fichier plus général sera toujours assigné à n'importe quel fichier qui correspond à un type de fichiers plus spécifique. Les types de fichiers cachés sont indiqués en gris clair.

Le bouton Déplacer pour révéler va déplacer un type de fichier caché plus haut dans la liste afin qu'il soit au dessous de tous les types de fichiers auxquels il correspond. Le résultat est que le type de fichier sera désormais activement utilisé par le schéma de couleurs. De manière similaire, le bouton Déplacer pour cacher déplace un type de fichiers actif plus bas dans la liste afin qu'il soit en dessous de tous les types de fichiers auxquels il correspond. (Ou au fond de la liste au cas ou il ne correspond à aucun autre type). Cacher un type de fichier signifie qu'il n'est plus utilisé dans le schéma de couleurs.

Les boutons Déplacer au dessus et Déplacer en dessous peuvent être respectivement utilisés pour bouger un type de fichier en haut ou en bas de la liste. Les boutons Déplacer en haut et Déplacer en bas peuvent être respectivement utilisés pour bouger un type de fichier au sommet où au fond de la liste. Vous pouvez également utilisez le "Glissez-Déposez" pour changer l'ordre des types de fichiers dans la liste.
