Interface de création

Quand l'application démarre, elle vous demandera de choisir un dossier. Après que vous ayez séléctionné ce dossier, ce dernier sera analysé afin de de déterminer la taille de tout les fichiers qui le composent (incluant les fichiers dans les sous-dossiers). Une fois l'analyse terminée, un "arbre carte" du dossier est affiché dans une nouvelle fenêtre. Once the scanning is finished a tree map of the scanned folder is shown in a new view window.

Il y a différentes manières de créer une fenêtre:

  • La commande Analyse Dossier... du menu "Fichier" vous permets d'analyser un autre dossier.
  • La commande Analyser de nouveau du menu "Fichier" vous permets de générer une version récente de la fenêtre actuelle. C'est utile quand le contenu du dossier a changé, par exemple, après que vous ayez supprimé un ou plusieurs fichiers. La nouvelle vue utilisera les mêmes préférences que la fenêtre courante.
  • La commande Dupliquer du menu "Fenêtre" vous permets de dupliquer la vue courrante. Une nouvelle fenêtre est crée à l'identique de la fenêtre courrante. Vous pouvez modifier les paramètres d'affichages afin d'afficher l'"arbre carte" de différente manière en même temps.
  • La commande Jumeaux du menu "Fenêtre" vous permets de créer une fenêtre jumelle de la fenêtre courrante. Exécuter cette commande est très similaire à la commande Dupliquer, hormis le fait que la navigation est également synchronisé.
  • La commande Filtre... du menu "Fenêtre" vous permets d'appliquer un filtre à la fenêtre courrante.
