Interface de création

Quand l'application démarrera, elle vous demandera de choisir un dossier. Après que vous ayez séléctionné ce dossier, ce dernier sera analysé afin de de déterminer la taille de tous les fichiers qui le composent (incluant les fichiers dans les sous-dossiers). Une fois l'analyse terminée, un "arbre carte" du dossier est affiché dans une nouvelle fenêtre.

Il y a différentes manières de créer une fenêtre:

  • La commande Analyse Dossier... du menu "Fichier" vous permets d'analyser un autre dossier.
  • La commande Analyse Filtrée du menu "Fichier" vous permet d'analyser un dossier tout en appliquant un filtre. De cette manière, vous pouvez, par exemple, exclure des sous dossiers qui ne vous intéressent pas. Cela peut améliorer de manière significative la rapidité d'éxécution de l'analyse.
  • La commande Analyser de nouveau du menu "Fichier" vous permet de générer une version récente de la fenêtre actuelle. C'est utile quand le contenu du dossier a changé, par exemple, après que vous ayez supprimé un ou plusieurs fichiers. La nouvelle vue utilisera les mêmes préférences que la fenêtre courante.
  • La commande Charger les données de l'analyse du menu "Fichier" vous permet de créer une vue avec les données d'analyse qui ont été sauvegardées plus tôt.
  • La commande Dupliquer du menu "Fenêtre" vous permet de dupliquer la vue courante. Une nouvelle fenêtre est créée à l'identique de la fenêtre courante. Vous pouvez modifier les paramètres d'affichage afin d'afficher l'"arbre carte" de différentes manières en même temps.
  • La commande Jumeaux du menu "Fenêtre" vous permet de créer une fenêtre jumelle de la fenêtre courrante. Exécuter cette commande est très similaire à la commande Dupliquer, hormis le fait que la navigation est également synchronisée.
  • La commande Filtre... du menu "Fenêtre" vous permet d'appliquer un filtre à la fenêtre courante.
