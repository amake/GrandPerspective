Interface de l'onglet Affichage

L'onglet Affichage dans le tiroir vous permets de modifier le rendu de l'affichage de l'interface. Vous pouvez choisir depuis de nombreuses options de coloration:

  • Niveau: Le nombre de sous dossiers dans le chemin de chaque fichier détermine la couleur, relativement au dossier qui est actuellement visible. Tous les fichiers qui ont le même "niveau" ont la même couleur.
  • Extension: Tous les fichiers qui ont la même extention ont la même couleur.
  • Dossier: Tous les fichiers dans le même dossier ont la même couleur.
  • Nom: Les fichiers avec le même nom auront la même couleur.
  • Rien: Tous les fichiers auront la même couleur.
  • Top Dossier: Le premier sous dossier dans le chemin de chaque fichier détermine la couleur, relativement au dossier qui est actuellement visible. Tout les fichiers qui appartiennent au même "top dossier" ont la même couleur.
  • type de fichier: le type de fichier de chaque fichier détermine sa couleur. Vous pouvez éditer l'ordre de la liste des types de fichiers pour controler quels types de fichiers sont associés à chaque couleur.

A côté de cela, vous pouvez choisir l'une des nombreuses palettes de couleurs qui sont fournies avec l'application. Choisissez en une qui convient à vos goûts. Il faut toutefois noter que le nombre de couleurs par palettes est variable. Ce qui a un effet sur la disposition des couleurs. Les couleurs dans une palette sont indiquées en dessous de cette dernière. Quand la disposition des couleurs utilisée possède des couleurs qui ont un sens significatif (actuellement, les schémas "Niveau" et "Type de fichiers"), une description est affichée à côté de chaque couleur.

Vous pouvez également configurer un masque. Quand vous utilisez un masque, seulement certains fichiers sont affichés. Les fichiers qui sont masqués continuent de prendre de la place dans la vue même si ces derniers ne sont pas affichés. Le masquage peut être utile pour identifier rapidement certain types de fichiers, également utile avant d'appliquer un filtre. Cliquez sur le bouton Edition pour éditer le masque.

Par défaut, la vue montre seulement les fichiers qui ont été analysés (ou un sous-ensemble). Toutefois, vous pouvez voir le volume en entier. En activant ce paramètre, la vue montrera également l'espace libre et l'espace utilisé de manière diverse. Notez que si la taille totale des fichiers analysés est petite par rapport à la taille totale du volume alors la vue ne sera pas très intéressante. Dans ce cas, vous devriez désactiver le paramètre "Montrer le volume en entier". Vous pouvez également analyser le dossier à la racine du volume, si vous ne l'avez pas déjà fait.

Vous pouvez configurer si le contenu des paquets doit être visibles ou non. Si le contenu est caché, chaque paquet apparaîtra alors dans la vue comme un seul fichier. Après avoir activé "Montrer le contenu des paquets", tous les fichiers dans les paquets seront visibles.
