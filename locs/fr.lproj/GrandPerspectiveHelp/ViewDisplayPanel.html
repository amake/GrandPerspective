Interface de l'onglet Affichage

L'onglet Affichage dans le tiroir vous permets de modifier le rendu de l'affichage de l'interface. Vous pouvez choisir depuis de nombreuses options de colorations:

  • Profondeur: Le nombre de sous dossiers dans le chemin de chaque fichier détermine la couleur, relativement au dossier qui est actuellement visible. Tout les fichiers qui ont la même "profondeur" ont la même couleur.
  • Extension: Tout les fichiers qui ont la même extension ont la même couleur.
  • Fichier: Tous les fichiers dans le même dossier ont la même couleur.
  • Nom: Les fichiers avec le même nom auront la même couleur.
  • Rien: Tous les fichiers auront la même couleur.
  • Top Dossier: Le premier sous dossier dans le chemin de chaque fichier détermine la couleur, relativement au dossier qui est actuellement visible. Tout les fichiers qui appartiennent au même "top dossier" ont la même couleur.

A côté de cela, vous pouvez choisir l'une des nombreuses palettes de couleurs qui sont fournies avec l'application. Choisissez en une qui convient à vos goûts.

Vous pouvez également configurer un masque. Quand vous utilisez un masque, seulement certain fichiers sont affichés. Les fichiers qui sont masqués continuent de prendre de la place dans la vue même si ces derniers ne sont pas affichés. Le masquage peut être utile pour identifier rapidement certain types de fichiers, également utile avant d'appliquer un filtre. Cliquez sur le boutton Edition pour éditer le masque.
