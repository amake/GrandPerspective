Sauvegarde des résultats de l'analyse

Il y'a deux manières pour sauvegarder le contenu de l'interface, elles sont toutes les deux accessibles depuis le menu "Fichier" :

  • La commande Sauvearde des données de l'analyse... vous permet de sauvegarder toutes les données dans l'interface. Les données sont sauvegardés dans un fichier XML. Le dossier visible dans l'interface ne gène en rien. Tous les fichiers (ainsi que ceux qui ne sont pas visibles dans l'interface) sont stockés. Toutefois, si un filtre a été utilisé pour créer la vue alors, les fichiers qui ont été exclus ne seront pas sauvegardés.

    Les données sauvegardées peuvent être chargées dans une nouvelle vue plus tard en utilisant la commande Charger les données de l'analyse.... La sauvegarde et le chargement des données peut être utiles à diverses raisons parmis les suivantes :

    • Vous pouvez voir et naviguer d'anciens dossiers ainsi que leur contenu, même si ces derniers ont désormais changé. Comparer l'ancien contenu d'un dossier avec l'actuel peut être utile pour déterminer pourquoi votre espace disque à soudainement disparu. Note : Si vous utilisez Time Machine, vous pouvez aussi voir l'ancien contenu des dossiers en analysant les anciennes sauvegardes de Time Machine,
    • Vous pouvez voir votre disque dur en utilisant GrandPerspective sur une autre machine.
    • Le chargement des données est plus beaucoup plus rapide que d'obtenir les mêmes données en analysant le dossier une nouvelle fois.
    • Les utilisateurs expérimentés peuvent transformer, analyser et manipuler les données sauvegardées en utilisant des scripts shell ou d'autres outils.

    Attention : Quand vous regardez des données qui ont été chargés, vous devez être très prudent en utilisant la commande Supprimer. Les fichiers ou dossiers que vous séléctionnez pour la suppression peuvent avoir un contenu différent de celui suggéré par la vue; les dossiers peuvent contenir plus d'un fichiers et les fichiers peuvent avoir changé (en taille) également.

  • La commande Sauvegarder l'image de la vue... peut être utilisé pour sauvegarder une image PNG de la fenêtre. L'image sera créée en utilisant les paramètres de l'onglet Affichage et montrera le même dossier. L'image est une représentation statique des données de la fenêtre; il est impossible de charger une image dans GrandPerspective pour de futures explorations. Vous pouvez toutefois charger l'image sur votre blog, l'envoyer par mail à vos amis ou encore l'encadrer et l'accrocher à votre mur :-).
