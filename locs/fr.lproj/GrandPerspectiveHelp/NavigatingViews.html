Interface de navigation

Vous pouvez afficher des informations supplémentaires sur les fichiers depuis l'interface principal en maintenant votre curseur au dessus du rectangle correspondant. Le fichier sous le curseur de la souris est séléctionné et marqué par un rectangle. Les dossiers se trouvent le fichier sont marqués de la même manière. Le nom et la taille du fichier séléctionné sont affichés en dessous de la vue principale.

Vous pouvez changer la séléction du fichier pour l'un des dossiers dans lequels il se situe (de cette manière, vous pouvez par exemple rapidement trouver la taille d'un dossier spécifique). Vous pouvez faire cela soit en utilisant votre molette de souris bien en utilisant les touches "[" et "]" de votre clavier. Après avoir bouger la séléction d'un niveau dans la hiérarchie des fichiers, il va rester à ce niveau jusqu'à ce que le fassiez monter ou descendre explicitement. Si vous ramenez la séléction en arrière (vers un fichier) et que vous allez un peu au-delà de cela, la séléction restera fidèle aux dossiers comme c'était le cas à l'origine.

Cliquer sur un fichier bloque le focus sur ce dernier afin que les informations ne changent pas quand vous déplacez votre souris, Cliquer une fois de plus, n'importe dans l'interface pour retirer ce "blocage".

Quand le focus est verrouillé, vous pouvez changer le dossier qui est affiché dans l'interface. En cliquant sur le boutton Bas, vous descendez dans le premier sous dossier qui est dans le chemin du fichier séléctionné. Vous pouvez continuer à cliquer sur ce boutton tant que vous n'avez pas atteint le fichier lui même. En cliquant sur Haut vous reviendrez à l'étape précédente, niveau par niveau.

Quand le focus est verouillé, vous pouvez voir le fichier séléctionné dans le Finder en utilisant le boutton Révéler. Une fois cela fait, vous pouvez renommer, supprimer le fichier, etc. (Toutes les actions qui sont possible dans le Finder.)
