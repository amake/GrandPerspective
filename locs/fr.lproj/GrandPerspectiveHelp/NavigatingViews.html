Interface de navigation

Vous pouvez afficher des informations supplémentaires sur les fichiers depuis l'interface principale en maintenant votre curseur au dessus du rectangle correspondant. Le fichier sous le curseur de la souris est séléctionné et marqué par un rectangle. Les dossiers où se trouvent le fichier sont marqués de la même manière. Le nom et la taille du fichier séléctionné sont affichés en dessous de la vue principale.

Changer le focus

Vous pouvez changer la séléction du fichier pour l'un des dossiers dans lequels il se situe (de cette manière, vous pouvez par exemple rapidement trouver la taille d'un dossier spécifique). Vous pouvez changer le focus de différente manière :

  • le contrôle du Focus dans la barre d'outils,
  • la molette de votre souris, ou bien
  • en utilisant les touches "[" et "]" de votre claiver.

Après avoir bougé la séléction d'un niveau dans la hiérarchie des fichiers, il va rester à ce niveau jusqu'à ce que vous le fassiez monter ou descendre explicitement. Si vous ramenez la séléction en arrière (vers un fichier) et que vous allez un peu au-delà de cela, la sélection restera fidèle aux dossiers comme c'était le cas à l'origine.

Vérouiller et dévérouiller la séléction

Cliquer sur un fichier verouille le focus sur ce dernier afin que les informations ne changent pas quand vous déplacez votre souris, Cliquer une fois de plus, n'importe où dans l'interface pour retirer ce verrou.

Zoom avant et arrière

Quand la séléction est vérouillé, vous pouvez zoomé en avant et en arrière en utilisant les contrôles de Zoom présent dans la barre d'outils. Cela changera le dossier qui est affiché dans la fenêtre principale. En zoomant en avant, vous descendez dans le premier sous dossier qui est dans le chemin du fichier séléctionné. Vous pouvez continuer à cliquer sur ce bouton tant que vous n'avez pas atteint le fichier lui même. En zoomant en arrière, vous reviendrez à l'étape précédente, niveau par niveau.

Actions

Quand la séléction est vérouillé, vous pouvez utiliser la barre d'outils pour appliquer les actions suivantes à vos fichiers (ou dossiers) séléctionnés :

  • Ouvrir ouvre les fichiers séléctionnés en utilisant l'applicatino par défaut qui y est associé dans le Finder.

  • Révéler affiche le fichier (ou dossier) séléctionné dans le Finder. Vous pouvez l'examminez, voir plus d'informations, le renommer, le déplacer, etc.

  • Supprimer déplace les fichiers (ou dossiers) séléctionnés vers la Corbeille. La vue sera mise à jour pour vous montrer l'espace ainsi libéré. Par défaut, le bouton est désactivé pour réduire les risques d'un geste maladroit. La suppression peut être activée depuis les Préférences.

    Attention : La suppression de dossier déplacera le dossier entier vers la Corbeille, cela inclut tous les fichiers présent dans le dossier qui ne sont pas visible dans la vue. Un fichier peut ne pas être visible car il est masqué ou filtré, il n'a pas pu être analysé pour cause de permissions insuffisantes ou il est apparu dans le dossier après l'analyse de ce dernier. En bref, vous devez faire très attention quand vous supprimez un dossier.
