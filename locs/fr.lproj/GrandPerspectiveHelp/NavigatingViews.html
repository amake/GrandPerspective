Interface de navigation

Vous pouvez afficher des informations supplémentaires sur les fichiers depuis l'interface principal en maintenant votre curseur au dessus du rectangle correspondant. Cliquer sur un fichier bloque le focus sur ce dernier afin que les informations ne changent pas quand vous déplacez votre souris. Cliquer une fois de plus, n'importe où dans l'interface pour retirer ce "blocage". Quand le focus est verrouillé, vous pouvez changer le dossier qui est affiché dans l'interface. En cliquant sur le boutton Bas, vous descendez dans le premier sous dossier qui est dans le chemin du fichier séléctionné. Vous pouvez continuer à cliquer sur ce boutton tant que vous n'avez pas atteint le fichier lui même. En cliquant sur Haut vous reviendrez à l'étape précédente, niveau par niveau.

Quand le focus est verouillé, vous pouvez voir le fichier séléctionné dans le Finder en utilisant le boutton Révéler. Une fois cela fait, vous pouvez renommer, supprimer le fichier, etc. (Toutes les actions qui sont possible dans le Finder.)
