Overzicht vensters cre‘ren

Wanneer het programma start toont het een Welkom scherm. Hier kunt u een map selecteren die eerder ingelezen is door deze te dubbelklikken in de lijst met recentelijk ingelezen mappen. Indien gewenst kunt u ook een andere map selecteren door Andere Map Inlezen te klikken. Nadat u een map geselecteerd heeft, wordt deze ingelezen om de grootte te bepalen van alle bestanden in de map (inclusief bestanden in deelmappen). Zodra het inlezen voltooid is, wordt een "boom diagram" weergave van de inhoud getoond in een nieuw overzicht venster.

Er zijn verschillende manieren waarop een nieuw overzicht venster gecre‘rd kan worden:

  • Via de Map inlezen opdracht in het "Archief" menu kunt u een nieuwe folder laten inlezen.
  • Via het sub-menu Recente map inlezen in het "Archief" menu kunt u snel een recentelijk ingelezen folder nogmaals laten inlezen. De map zal worden ingelezen zonder filter, onafhankelijk van de manier waarop de folder eerder ingelezen was.
  • Via de Map inlezen met filter opdracht in het "Archief" menu kunt u een nieuwe folder laten inlezen en gelijktijdig een filter toepassen. Op deze manier kunt u bijvoorbeeld mappen overslaan waarin u toch niet geinteresseerd bent, wat het inlezen aanzienelijk kan versnellen.
  • Via de Map herlezen opdracht in het "Archief" menu kunt u een bijgewerkte versie cre‘ren van een bestaand venster. Dit is nuttig wanneer de inhoud van een map veranderd is, bijvoorbeeld nadat u ŽŽn of meerdere bestanden heeft verwijderd. Het nieuwe venster zal dezelfde instellingen gebruiken als het venster waarop het gebaseerd is.
  • Via de Ingelezen data laden opdracht in het "Archief" menu kunt u eerder opgeslagen data opnieuw bekijken.
  • Via de Dupliceer opdracht in het "Venster" menu kunt u een venster dupliceren. Het nieuwe venster zal lijken op het bestaande venster. Vervolgens kunt u de weergave instellingen wijzigen zodat u tegelijkertijd dezelfde informatie op verschillende manieren kunt bekijken.
  • Via de Maak tweeling opdracht in het "Venster" menu kunt u een venster dupliceren, met een blijvende koppeling naar een ander venster. Deze koppeling zorgt dat de navigatie binnen beide vensters volledig gesynchroniseerd is.
  • Via de Filter opdracht in het "Venster" menu kunt u een filter toepassen op de inhoud van een venster.

In uw Voorkeuren kunt u instellen wat er moet gebeuren nadat u het laatste overzicht venster heeft gesloten:

  • Toon het Welkom scherm weer
  • Sluit het programma automatisch
  • Geen van beide
