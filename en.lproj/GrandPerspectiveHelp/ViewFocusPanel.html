The Focus tab

The Focus tab in the Control Panel shows the name and size of the folder that is currently shown in the active view window. This is not necessarily the folder that was scanned, because you can navigate down into subfolders.

The path of the file (or folder) that is currently selected is also shown, together with its size. The exact size is shown, together with an approximation that is easier to interpret. When the selected item is a hard link, the size is shown in dark-gray as opposed to black. When the selected item is a file, its file type is shown. Finally, the panel shows the creation, modification and access time of the selected item.
