How to edit or add new color palettes

You cannot edit the color palettes in the Display tab from within GrandPerspective. However, the color palettes are stored in the standard Mac color list file format so you can edit the palettes and add new ones using a color list editor that supports this format. An example is ColorListEditor provided by Modesitt Software. When you store the new palette inside the GrandPerspective palettes folder (GrandPerspective.app/Contents/Resources/Palettes) you can use it the next time you run GrandPerspective.
