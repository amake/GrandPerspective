The Control panel

The Control panel can be used to inspect and control the active view window. It can be opened in two ways:

  • via the Info button in the toolbar of each view window, and
  • by selecting "Show control panel" in the "Window" menu.

The control panel contains three tabs:

  • The Info tab gives information about the folder that has been scanned.
  • The Display tab lets you configure how the view is displayed.
  • The Focus tab shows details about the folder that is currently shown in the view and the file that is in focus.
