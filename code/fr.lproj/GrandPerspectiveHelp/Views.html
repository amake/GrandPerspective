Interface de GrandPerspective

GrandPerspective visualise l'utilisation du disque en utilisant un "arbre carte". Une fenêtre de l'interface montre le contenu d'un dossier donné. Chaque rectangle représente un fichier dans le dossier. L'aire du rectangle est proportionnel à la taille du fichier. Les fichiers qui sont dans le même dossier apparaissent ensemble mais leur placement est arbitraire. Vous pouvez vous concentrer sur un fichier en laissant le pointeur de la souris au dessus du rectangle de ce dernier. Le nom du fichier, avec sa taille, est alors affiché dans le bas de la fenêtre principal.
