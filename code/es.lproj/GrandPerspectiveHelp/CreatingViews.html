Creando vistas

Cuando la aplicación comienza, te pregunta para que selecciones una carpeta. Después de haber seleccionado una carpeta, la carpeta es escaneada para determinar el tamaño de todos los archivos en ella (incluyendo los archivos en subcarpetas). Una vez el escaneo a terminado un mapa de árbol de la carpeta escaneada es mostrada en una nueva ventana de vista.

Hay otroscaminos variado con los que puedes crear una vista:

  • El comando Escanear carpeta... del menú "Archivo" te permite escanear otra carpeta.
  • El comando Re-escanear del menú "Archivo" te permite generar una versión actualizada de la vista actual. Esto es útil cuando el contenido de la carpeta ha cambiado, por ejemplo porque as borrado uno o más archivos. La nueva vista usará la misma configuración que la vista actual.
  • El comando Duplicar del menú "Ventanas" te permite duplicar la ventana actual. Una nueva vista es creada similar a la actual. Puedes cambiar la configuración de visualización para ver el mapa de árbol de varias formas diferentes a la vez.
  • El comando Clonar del menú "Ventanas" te permite clonar la ventana actual. Clonar una vista es muy similar a duplicar la, excepto que la navegación está sincronizada también (lo que hagas en una ocurre en la otra).
  • El comando Filtrar... del menú "Ventanas" te permite aplicar un filtro a la vista actual.
