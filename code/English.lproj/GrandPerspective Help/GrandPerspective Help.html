GrandPerspective Help

Introduction

GrandPerspective is a utility application that graphically shows the disk usage within a file system. It can help you to manage your disk, as you can easily spot which files and folders take up the most space. It uses a so called tree map for visualisation. Each file is shown as a rectangle with an area proportional to the file's size. Files in the same folder appear together, but their placement is otherwise arbitrary.

Viewing files and folders

When the application starts it prompts you to select a folder. After you have selected a folder, the application will scan the folder to determine the size of all files that it contains (also the files that in sub-folders). Once the scanning is finished, which may take a short while when the folder contains a large number of files, a tree map of the scanned folder is shown in a new view window. Each rectangle in the window represents a file in the folder. You can focus on a file by hovering the mouse pointer over it. The name of the file, together with its size, is then shown at the bottom of the window.

Clicking on a file locks the focus, and clicking once anywhere in the view, unlocks the focus again. When the focus is locked, you can change the contents of the view. By clicking the Down button, you descend into the first subfolder of the file's path. You can keep going down, until you've reached the file itself. Clicking Up will take you back up, one level at the time. When the focus is locked, you can show the file in Finder using the Reveal button. You can then subsequently examine the file in more detail and rename it, or delete it, should you wish to.

More about the view

At the bottom right of the GrandPerspective view window there is a More... button. Click on it, and a drawer will appear (click once more, and it disappears again). The drawer contains three panels described next.

The Info panel

The Info panel gives information about the folder that has been scanned. It shows the name of the folder, and also the time when it was scanned. When a filter has been applied, it will also show details of this. It also shows which file size measure has been used, and the total size of the folder is scanned.

The Display panel

The Display panel enables you to change how the view is drawn. You can choose from various options that determine the coloring for files:

  • Depth: Files at the same level in the **
  • Extension: All files with the same extension have the same color.
  • Folder: All files in the same folder have the same color.
  • Name: Files with the same name will have the same color.
  • Nothing: All files will have the same color.
  • Top Folder: All files that appear in the same "top folder" have the same color. Here the top folders are the subfolders that appear directly in the folder that is currently shown in the view. (Just try it).

Next to that, you can also select on of the various color palette that is provided with the application. Just pick one that you suits your taste.

You can also configure a mask. When you use a mask, only certain files are shown. Files that are masked, on the other hand, still take up space but are not drawn. Masking can be useful for quickly identifying certain (types of) files, and can also be useful step before applying a filter. Press the Edit button to start editing the mask,

The Focus panel

The focus panel shows the name and size of the folder that is currently shown in the view. This is not necessarily the folder that was scanned, because you can navigate down into subfolders.

The path of the file that is currently in focus is also shown, together with its size. Even though this is information is also displayed at the bottom of the view, it is repeated here as long paths may not entirely fit under the view. Furthermore, here you can find the exact size.

Creating new views

There are various ways in which you can create new views:

  • The Scan Folder... command from the "File" menu lets you scan another folder.
  • The Re-scan command from the "File" menu lets you generate an up-to-date version of the current view. This is useful when the folder contents have changed, for example because you deleted one or more files. The new view will use the same settings as the current view.
  • The Duplicate command from the "Window" menu lets you duplicate the current view. A new view is created similar to the current one. You can subsequently change its display settings to see the information in different ways at the same time.
  • The Twin command from the "Window" menu lets you twin the current view. Twinning a view is very similar to duplicating it, except that navigation is synchronized as well.
  • The Filter... command from the "Window" menu lets you apply a filter to the current view.
